# Lastr'

**Take control. Last longer.**

Lastr' is a men's sexual health app that helps users overcome premature ejaculation through a 90-day, science-backed training program. It combines daily exercises, habit-building, progress tracking, and personalized routines into a discreet mobile experience users can do on their own.

## Features

- **90-day program** — structured, progressive training plan
- **Daily exercises** — guided routines with timers and instructions
- **Progress tracking** — streaks, stats, and milestones
- **Personalization** — onboarding tailors the program to the user
- **Subscriptions** — premium tier via RevenueCat
- **Auth** — email + Apple sign-in via Supabase

## Tech

## Stack

- **Expo** ~52 / **React Native** 0.76
- **Expo Router** for file-based navigation
- **Supabase** for auth and backend
- **RevenueCat** for subscriptions / IAP
- **Zustand** for state management
- **TypeScript**

## Getting started

```bash
npm install
npm start
```

Then run on a device/simulator:

```bash
npm run ios       # iOS
npm run android   # Android
npm run web       # Web
```

## Scripts

- `npm start` — start the Expo dev server
- `npm run ios` / `android` / `web` — run on platform
- `npm run lint` — run ESLint
- `npm test` — run Jest
