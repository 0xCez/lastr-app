# lastr-app

A React Native (Expo) mobile app built with Expo Router, Supabase auth, and RevenueCat in-app purchases.

## Stack

- **Expo** ~52 / **React Native** 0.76
- **Expo Router** for file-based navigation
- **Supabase** for auth and backend
- **RevenueCat** for subscriptions / IAP
- **Zustand** for state management
- **TypeScript**

## Getting started

```bash
npm install
npm start
```

Then run on a device/simulator:

```bash
npm run ios       # iOS
npm run android   # Android
npm run web       # Web
```

## Scripts

- `npm start` — start the Expo dev server
- `npm run ios` / `android` / `web` — run on platform
- `npm run lint` — run ESLint
- `npm test` — run Jest
